Inicios de html

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, trabajador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de hipertexto para compartir documentos.

El hipertexto permitía que los usuarios accedieran a la información relacionada con los documentos electrónicos que estaban visualizando, estándarespodrían asimilarse a los enlaces de las páginas web actuales. presentó a una convocatoria organizada para desarrollar un sistema de "hipertexto" para Internet. Después de unir sus fuerzas con el ingeniero de sistemas Robert Cailliau, presentaron la propuesta ganadora llamada WorldWideWeb (W3).

Para la definición del estándar HTML, Tim Berners-Lee se basó en el lenguaje de marcado SGML (Standard General Markup Language). Este lenguaje define reglas de etiquetado y estructura generales. A partir de SGML se han definido lenguajes como HTML, Postscript, RTF,…

Historia de HTML

  1. En 1980, Tim Berners-Lee del CERN propuso un sistema de hipertexto para compartir documentos, dando origen a la historia de HTML.
  2. En 1991, Berners-Lee publicó la primera descripción formal de HTML llamada "HTML Tags".
  3. En 1993, se presentó la primera propuesta oficial para estandarizar HTML por parte del IETF.
  4. En 1995, se publicó el estándar HTML 2.0 por parte del IETF.
  5. A partir de 1996, el W3C comenzó a publicar estándares de HTML, con HTML 3.2 en 1997, que incorporaba avances como applets de Java y texto alrededor de imágenes.
  6. El 24 de abril de 1998, se lanzó HTML 4.0, que introdujo hojas de estilo CSS, scripts en páginas web, mejoras en accesibilidad, tablas complejas y mejoras en formularios.
  7. La última especificación oficial, HTML 4.01, se publicó el 24 de diciembre de 1999, sin cambios significativos.
  8. A partir de HTML 4.01, el W3C se centró en el desarrollo de XHTML, lo que llevó a la formación del grupo WHATWG en 2004 debido a la falta de interés del W3C en HTML.
  9. En 2008, se publicó el primer borrador oficial de HTML5 por parte de WHATWG.
  10. En 2014, HTML5 se convirtió en un estándar recomendado por el W3C, permitiendo el desarrollo de aplicaciones multiplataforma junto a Javascript y CSS3.

Algunas de las etiquetas básicas de HTML son las siguientes:

  • <var>: Nombre de variable utilizado en contexto matemático o de programación.
  • <pre>: Texto preformateado.
  • <dd>: Definición/Descripción de un término en la lista de descripciones.
  • <dt>: Término en la lista de descripciones.
