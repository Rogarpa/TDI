Historia del Css

Cronología

Versión Fecha Descripción Características inovadoras
CSS 1 1996 Lanzado incompleto, terminó por relanzarse el 11 de enero de 1999 con correcciones.
  • Añade estilo a html en un documento por separado
CSS 2 12 mayo de 1998 Basado en CSS1,
  • Añade soporte para diferentes medios de salida.
CSS 2.1 2 de agosto de 2002 Supuso una modularización del mismo CSS, para que cada modulo pudiera desarrollarse independientemente del resto de módulos.
Permitiendo que nuevos modulos se crearan definiendo nuevas funcionalidades o extendiendo anteriores de forma completamente independiente.
  • Capítulos
  • Modularización
CSS 3 12 April 2016 Se denomina como tal a todas las modificaciones posteriores a CSS 2.1.
  • Grids
  • Blend mods
  • Masks
  • Shapes
