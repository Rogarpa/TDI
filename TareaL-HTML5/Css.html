Historia del Css

Definición

Es un lenguaje de hojas de estilo usado para describir la presentación de documentos escritos en lenguajes como HTML o XML. Te descirbe como los elementos deben renderizarse en pantalla, u otros medios.
CSS está entre los lenguajes claves de la web abierta, bajo los estándares de las especificaciones de la W3C. Antes el desarrollo de las varias partes de la especificación del Css era hecha sincorna... Y a pesar de que CSS1, CSS2.1 y CSS3 fueron versiones lanzadas oficialmente, ya no existirán versiones más nuevas debido a que de ahora en adelante solo se le llama CSS.
Luego de CSS 2.1, el alacance de la especificación incrementó y el desarrollo de los diferentes módulos comenzó a diverger, que pensaron que era más eficiente lanzar recomendaciones por separado dependiendo del módulo. Hoy en día dejaron de lado el versionado de la especificación de CSS, y ahora la W3C periodicamente toma instantaneas del último modulo estable de CSS y modulos individuales, de forma que los modulos si tienen números de versión o niveles.

Historia previa

  • Octubre de 1994 el gramático noruego Håkon Wium Lie propone la idea de Css.
  • A la próxima conferencia en abril de 1995, Css se presenta de la mano de Bert y Håkon.
  • El primer navegador comercial en soportar CSS fue Internet Explorer 3, lanzado en agosto de 1996 soportando la mayoría de propiedades tales como color, background fuente y propiedades de texto; pero no implementaba mucho de las opciones del modelo de caja.
  • February 1997, CSS got its own working group inside W3C and the new group set out to work on the features which CSS1 didn't address.
